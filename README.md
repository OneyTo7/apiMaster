# API Master

API Master是一款一体化的API设计、测试和协作平台，专注于提升开发者效率，简化API全生命周期管理。

## 项目结构

- `backend/` - 后端服务
- `frontend/` - 前端应用
- `docs/` - 项目文档

## 技术栈

- **后端**: Java + Spring Boot + MySQL + Redis
- **前端**: Vue3 + TypeScript + Element Plus

## 快速开始

### 后端

```bash
cd backend
# 构建项目
mvn clean package
# 运行项目
java -jar target/api-master-backend.jar
```

### 前端

```bash
cd frontend
# 安装依赖
npm install
# 开发模式运行
npm run dev
# 构建生产版本
npm run build
```
